# Calculadora-Funcional
🧮 Calculadora web desenvolvida com HTML, CSS e JavaScript. O projeto realiza operações matemáticas básicas e cálculo de porcentagem, sendo possível calcular valores como 10% de forma direta. Criado com foco em prática de lógica de programação, responsividade e manipulação do DOM.

## 🚀 Funcionalidades

- ➕ Adição
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- 📊 Cálculo de porcentagem (ex: 10% = 0.1)

## ⚠️ Melhorias futuras

- Implementar porcentagem em operações (ex: 10 + 10%)
- Melhorar design da interface
- Suporte ao teclado

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
